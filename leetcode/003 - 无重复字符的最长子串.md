## 第一次

i, j 作为滑动窗口的两端，map 作为 hash，用来查找窗口内是否有重合。

```c++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        map<char, int> hash;
        int size = s.size();
        int ret = 0;
        int i, j = 0;

        for (; j < size; ++j) {
            auto it = hash.find(s[j]);
            if (it != hash.end()) {
                int idx = it->second;
                if (idx >= i)
                    i = idx + 1;
            }

            hash[s[j]] = j;

            if (j - i + 1 > ret) {
                ret = j - i + 1;
            }
        }

        return ret;
    }
};
```

评测结果 56 ms + 8.5 MB。
